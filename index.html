El sector informático está en constante evolución y por tanto, los lenguajes de programación varian según las necesidades del momento. Por ello, me ha parecido interesante ver qué lenguajes de programación eran los más usados en 2021.

Frameworks de JavaScript

Dentro de JavaScript hay multitud de frameworks. En este gráfico nos hemos quedado con los 4 más importantes y que más usuarios tienen en 2021, así lo demuestran las búsquedas en Google. Gracias a la herramienta Google Trends podemos saber la notoriedad de cualquier búsqueda que realizan los usuarios en un lapso de tiempo que definimos a placer.

Angular

Angular es un framework para aplicaciones web desarrollado en TypeScript, de código abierto, mantenido por Google, que se utiliza para crear y mantener aplicaciones web de una sola página. Su objetivo es aumentar las aplicaciones basadas en navegador con capacidad de Modelo Vista Controlador (MVC), en un esfuerzo para hacer que el desarrollo y las pruebas sean más fáciles.

Empresas importantes que utilizan Angular:

  • Google.
  • Wix.

React

React es una biblioteca Javascript de código abierto diseñada para crear interfaces de usuario con el objetivo de facilitar el desarrollo de aplicaciones en una sola página. Es mantenido por Facebook y la comunidad de software libre. En el proyecto hay más de mil desarrolladores libres.

Empresas importantes que utilizan React:

  • Facebook.
  • Instagram.
  • Tesla.
  • Discord.

Vue.js

Vue.js es un framework de JavaScript de código abierto para la construcción de interfaces de usuario y aplicaciones de una sola página. Fue creado por Evan You, y es mantenido por él y por el resto de los miembros activos del equipo central que provienen de diversas empresas como Netlify y Netguru.

Empresas importantes que utilizan Vue.js:

  • Nintendo.
  • Adobe.
  • Gitlab.

Node.js

Node.js es un entorno en tiempo de ejecución multiplataforma, de código abierto, para la capa del servidor (pero no limitándose a ello) basado en el lenguaje de programación JavaScript, asíncrono, con E/S de datos en una arquitectura orientada a eventos y basado en el motor V8 de Google. Fue creado con el enfoque de ser útil en la creación de programas de red altamente escalables, como por ejemplo, servidores web. Fue creado por Ryan Dahl en 2009 y su evolución está apadrinada por la empresa Joyent, que además tiene contratado a Dahl en plantilla.

Empresas importantes que utilizan Node.js:

  • LinkedIn.
  • Netflix.
  • Paypal.

Análisis del gráfico

Empecemos aclarando qué es la notoriedad. Los números reflejan el interés de búsqueda en relación con el valor máximo de un gráfico en una región y un periodo determinados. Un valor de 100 indica la popularidad máxima de un término, mientras que 50 y 0 indican que un término es la mitad de popular en relación con el valor máximo o que no había suficientes datos del término, respectivamente.

Una vez que sabemos interpretar los datos de la tabla, podemos ver como React es el framework que más ha crecido desde su creación. Durante mucho tiempo Angular fue el más popular de los frameworks de JavaScript pero en los últimos años React se ha convertido en el más usado.

¿Esto quiere decir que es el mejor framework? No. Todos los frameworks tienen su sentido y utilidad. Dependerá de qué aplicación o web queramos realizar para saber cual de ellos es el más apropiado y al que más partido le sacaremos.

¿Es JavaScript un buen lenguaje para backend?

JavaScript es un lenguaje que se puede utilizar tanto en frontend como backend. Aunque no sea la opción más utlizada a nivel global, más de un 8% de desarrolladores piensan que los frameworks de JavaScript para backend son la mejor solución para sus aplicaciones o webs.

Me ha parecido interesante saber si JavaScript es un lenguaje utilizado sólo para la parte más visual o también se utiliza para la parte que el usuario no ve, el backend. No ha sido fácil acceder a estos datos pero después de una intensiva búsqueda he dado con 2 páginas que muestran datos.

Los datos de 2019 los he encontrado en mxx.news mientras que los datos de 2020 los he podido sacar de darly.solutions. Gracias a ambos estudios podemos ver la evolución de los distintos lenguajes usados para backend.
